# LIS-001 — LAWRENCE Interaction System

**Status:** Accepted · v1 (foundation + runtime-console application)

The interaction layer that makes LAWRENCE feel like a governed operating system,
not a dashboard. Every interaction proves the runtime is **executable and
governed**: an action shows its intent before it runs, runs optimistically, and
stays reversible.

## Principles

1. **Show intent before execution.** No governed surface acts as a black box.
   Hovering reveals intent, authority, risk tier, evidence, and expected result.
2. **Optimistic, never blocking.** A click registers instantly (`Requested`),
   passes a visible governance gate (`Awaiting governance`), then settles
   (`Approved` / `Escalated` / `Dismissed`). No bare spinner ever blocks intent.
3. **Reversible.** Every settled decision exposes an `Undo` window — the UI
   mirrors the kernel's reversibility guarantee.
4. **Tokens, not magic numbers.** Durations, easings, focus rings, state colors,
   and elevation are CSS variables (`--lis-*`). Nothing hardcodes a duration.
5. **Scoped.** Everything lives under `.eor` and `lis-*` so the system never
   leaks into the rest of the app, and honors `prefers-reduced-motion`.

## Foundation

| Artifact | What it provides |
| --- | --- |
| `src/components/runtime-console/lis.css` | Motion/easing/state/focus/elevation tokens + primitive styles |
| `src/components/runtime-console/lis.tsx` | `useGovernedAction`, `GovernedActionBar`, `IntentPreview`, `ContextZone` + `useContextMenu` (`LisMenuProvider`), `useClipboard` |
| `src/lib/runtime-console/data.ts` | `IntentMeta` governance metadata (`OBJECT_INTENT`, `ACTION_INTENT`, `CAPABILITY_INTENT`) + copy payload helpers |

## Primitives (v1, in priority order)

1. **Optimistic feedback** — `useGovernedAction` / `GovernedActionBar`.
   `approve → requested → governing → granted`, `escalate → … → escalated`,
   `dismiss → requested → applied`. All reversible while settled.
2. **Hover intent previews** — `IntentPreview` renders a hovercard with
   intent / authority / risk / evidence / expected result for any governed
   object, action, or capability.
3. **Universal copy / context menu** — `ContextZone` + `useContextMenu` give
   every object surface a right-click menu: Copy link, Copy JSON, Copy ID,
   Open in Workspace, Pin.

## Deferred (v2)

Multi-select action toolbar and systematic microanimations (count-ups,
pipeline pulse, cursor-following glow) are intentionally deferred until the
runtime's execution semantics are stable.
