# Operator Training
- **Command Center** is the one place to act: action/review/risk/recommendation/notification/incident queues + learning signals.
- Run functions/agents; review grounded answers with citations.
- Approve/reject review cases and approval requests.
- Capture feedback (useful / incorrect / bad citation / correction) — feeds governed learning signals.
- Record recommendation outcomes (accept/reject/modify/escalate).
- Watch observability for cost/latency/failures; eval results for quality.
