# Success Metrics (by pack)
- **Recruiting**: time to shortlist, submission quality, interview conversion, evidence coverage, review override rate.
- **Onboarding**: Day-1 readiness, overdue owner tasks, missing docs, escalation rate.
- **Support**: answer confidence, citation coverage, time to draft, review rate, knowledge gaps.
- **Claims**: validation cycle time, missing evidence rate, contradiction rate, manual review rate.
- **Executive**: risk resolution time, approval cycle time, margin leakage prevented, decision memo throughput.
- **Healthcare**: authorization gap rate, care gap closure time, compliance findings resolved.
- **Professional Services**: margin leakage prevented, validation cycle time, delivery issue escalation rate.
