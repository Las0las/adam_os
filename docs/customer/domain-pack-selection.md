# Domain Pack Selection
Bundles (see `/domain-packs` and `src/lib/setup/default-domain-pack-plan.ts`):
- **Staffing / Recruiting OS**: recruiting + onboarding + executive-commercial
- **Support OS**: support + executive-commercial
- **Claims Validation OS**: claims + executive-commercial
- **Professional Services OS**: professional-services + onboarding + executive-commercial
Healthcare Ops is available as a standalone pack. Each pack ships objects, workflows,
guided demos, eval suites, and customer-readiness assets.
